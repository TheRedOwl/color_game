# Verziókezelés


Parancssorban PAT-al:

-Helyi repository inicializálása:
    -**git init**

-Projekt könyvtár ellenőrzése:
    -**git status**

-Ha van elmenteni valónk akkor előkészítjük a színpadra (stage):
    -**git add .**

-Ellenőrzés:
    -**git status**

-Elmentjük a helyi repóba:
    -**git commit -m "First commit"**
-Ellenőrzés:
    -**git status**
